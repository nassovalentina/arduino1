Arduino

Arduino es una plataforma de hardware y software libre que facilita la creación de proyectos electrónicos interactivos. Se basa en una placa con un microcontrolador y un entorno de desarrollo integrado (IDE) en el que se puede escribir código para controlar dispositivos físicos, como sensores y motores. Es una herramienta accesible, diseñada para que personas con y sin experiencia puedan construir proyectos como robots o sistemas de domótica.
